Scenario: Login as an existing user

 User are allowed to connect if they have an account

Scenario:
 Given the user enters a registered email
 And the user enters the password of the email
 When the user connects
 Then the user is connected to their account

 Scenario:
  Given the user enters an unregistered email or an incorrect password
  When the user connects
  Then the user is notified that their credentials are invalid
  And the user is not connected
  And the credentials are cleared

Scenario:
 Given the user checks "Remember me"
 And credentials are correct
 When the user connects
 Then the user is connected
 And will not have to login next time
